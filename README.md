Hi 👋 — I'm a junior software engineer based in São Paulo. I'm still in my
degree, so most of what lives on this profile is coursework. I'm keeping it
public because it's part of the story, not because I think it's impressive.

What I actually spend my time on right now is a project called **Kyron**.

---

## 🏋️ Why Kyron exists

I train across a few different sports and I could never find an app that
generated training plans I'd actually follow. Everything out there is either
locked to one discipline, or so generic I'd do better writing plans in a
notebook myself. So I'm building the thing I wanted to use.

Kyron is a mobile app that uses LLMs to do the planning work a human coach
would normally do. The app itself is private while I figure it out, but the
landing page is public:

## 🧪 Why Flutter?

Honestly? Because I'd never touched it before. I wanted Kyron to force me to
learn a whole new stack from scratch instead of reaching for something I
already knew. Dart and Flutter are the first tools on this profile where I'm
learning the whole thing as I go — not finishing a homework.

It's slower. I break things I wouldn't break in TypeScript. That's the point.
